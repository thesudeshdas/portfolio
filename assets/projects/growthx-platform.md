Owned frontend and backend development across 6 repositories for GrowthX — a B2B community platform for professionals.

Built the end-to-end event lifecycle: creation, registration with approval workflows, waitlist handling, post-registration experience with animated confirmation UI, venue details, calendar integration (Google, Outlook, ICS), event replay, photo albums, and attendee management.

Developed the job board with listings, application tracking, and external job link redirection. Built the admin panel with talent management dashboards, static pages CMS with live HTML preview, and a template-driven landing page builder with 8+ configurable sections.

Implemented platform-wide infrastructure: unified navigation system (sidebar, top nav, bottom nav), dynamic SEO (programmatic robots.txt, city-based sitemaps, OG meta tags), route-level access control, and analytics integration (Amplitude, Sentry) across 10+ feature modules.

On the backend, built and maintained 8 microservices in a TypeScript/Express/MongoDB/Redis monorepo managed with Turborepo, including a notification microservice using RabbitMQ with TTL + Dead Letter Exchange for delayed delivery across WhatsApp, SMS, and voice channels.
