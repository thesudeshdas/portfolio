Built and evolved TalentPlace.ai's flagship Career Profile Builder frontend
across the original app, a TypeScript rewrite and later monorepo edition.

The product included multi-step onboarding, validated profile section editors,
drag-and-drop section reordering, frontend auth, account lifecycle flows,
personality-assessment report UI, subscription flows and CRUD-heavy profile
management.

The work was frontend-owned: React, TypeScript, Chakra UI, Redux Toolkit,
React Hook Form, yup, React Router and API integration from the frontend side.
