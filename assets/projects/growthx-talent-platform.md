Built the complete B2B talent platform for GrowthX partner companies.

Developed the job management system from scratch: job posting with rich text editor, job verification workflow with admin approval/rejection, job lifecycle states (active/inactive/archived), and AI-powered job autofill from URLs with timeout handling and feature flagging.

Built the member directory with job-based filtering, profile navigation, candidate curation and shortlisting, and bulk download actions. Implemented the team invite workflow with invite modals, invite code validation, persistent banners, and auto-opening registration.

Created the landing page with hero section, marquee animations, and pricing page with FAQ. Migrated server state from Redux thunks to RTK Query, reducing redundant API calls.
