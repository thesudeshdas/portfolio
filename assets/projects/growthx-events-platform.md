Built GrowthX events platform workflows across web, mobile, backend and admin
surfaces.

The work covered event discovery, registration, approval flows, activity
feeds, galleries, event chat and operational tooling. Verified production
scale includes 1,052 events held, 23,110 registrations and 9,798 unique
registrants.

The real-time chat system is a dedicated subsystem of this work, but the
events platform itself spans the full user journey from discovery through
post-event engagement and operations.
